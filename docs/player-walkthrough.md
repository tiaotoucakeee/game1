# 《不存在的学生》玩家攻略

> 面向首次游玩的新玩家：从零开始，如何调查「程野」之谜，并抵达**真结局**。  
> 建议游玩时长：40–60 分钟。建议使用电脑浏览器，并**记录已用过的搜索词与访问过的页面**。

---

## 一、这是什么游戏？

这是一款以**网页搜索与信息核对**为核心的 ARG（平行实境游戏）。你扮演祁玉——2036 年暑假协助学院核对优秀毕业生档案的临时审核员。你在邮箱、学院官网、审核工作台、学生个人系统之间切换，通过**搜索关键词**与**核对档案**找出异常，最终在提交页作出判断。

**没有传统关卡菜单。** 页面不会直接告诉你「下一步去哪」；线索藏在正文、项目编号、人名与反复出现的词语里。

## 四、解锁审核终端（提交前必做）

仅收齐线索还不够：必须让工作台知道你在名单里发现了异常。

1. 回到 **`/audit/graduates`**。
2. 找到 **程野**，在操作中点击 **「信息有误」**（不要选「核查无误」）。
3. 右下角 **审核终端** 启动；之后 **`/audit/submit`** 才会出现完整提交选项。

若从未标记程野异常就提交，只能走 **「信息无误」** → 坏结局《无事发生的暑假》。

## 五、提交与反思问卷（结局分叉点）

### 5.1 提交页 `/audit/submit`

核心线索 **8/8** 且终端已解锁时，会出现两个意见：

| 选项 | 结果 |
|------|------|
| **更正为创作路径账户** | 进入全屏 **5 道反思题** → 再分真/坏结局 |
| **按异常记录清理**（删除申请） | 确认后 → 坏结局《不存在的人》 |

### 5.2 反思问卷 `/audit/reflection`

黑底白字，共 5 题。**前 4 题任意选择，不影响结局**；**第 5 题唯一分叉**：

> 综合全部材料，你是否认可「程野」作为创作路径账户（CYA-0000）继续留存于系统中？

| 第 5 题 | 条件 | 结局 |
|---------|------|------|
| **不认可** | — | 坏结局二《不存在的人》`/audit/delete` |
| **认可** | 未从 Ani 获得招募码 | **真结局一** |
| **认可** | 已从 Ani 获得招募码 | **真结局二** |

## 六、真结局一：《被看见的路径》

**条件：** 8 条核心线索 + 终端已解锁 + 提交「更正为创作路径账户」+ 反思 **认可** + **未**持有招募码（`open_path_code`）。

**推荐路线（不主动走 Ani 招募线）：**

1. 完成第三、四节全部步骤。
2. **不要**在学生系统向 Ani 问到招募码 `PATH-2034-LC`（或未打开 Ani 对话亦可）。
3. `/audit/submit` → **更正为创作路径账户** → 反思 Q5 **认可**。
4. 全屏阅读 **`/audit/truth`**（黑白渐变文字演绎 + 后日谈）。
5. 滚至底部，点击 **Stroll** → **`/audit/stroll`** 尾飞演出。
6. 学院 logo 出现后约 5 秒，点击 **返回邮箱** → 收件箱新邮件：**制作组感谢信**。

## 七、真结局二：《下一段路径》

**条件：** 在真结局一基础上，额外完成 **Ani 对话招募线**，并在反思 Q5 仍选 **认可**。

### 7.1 额外步骤：学生系统 + Ani AI

1. **`/student`** 用祁玉学号登录。
2. 首页右下角打开 **Ani AI** 对话（或访问 `/student/home?chat=ani`）。
3. 建议对话方向（演示模式与 Coze 均可用类似提问）：
   - 提及自己的毕设被多次退回、跨动画/游戏/交互方向；
   - 询问林澈、Creative Yard、跨专业创作案例；
   - 明确问：**「我想加入创作路径计划，招募码是什么？」**
4. 当 Ani 回复中出现 **`PATH-2034-LC`** 时，自动获得隐藏线索 **开放创作路径招募码**。
5. （可选）访问 **`/student/project`** 查看祁玉毕设三次退回 → 附加线索「个人毕业项目多次退回」。

### 7.2 真结局二演出流程

1. 完成真结局一所需调查 + **Ani 招募码**。
2. `/audit/submit` → 路径意见 → 反思 Q5 **认可**。
3. **`/audit/path-invite`**：CYA 科技绿内推页，招募码自动流式填入 → 点击 **加入创作路径团队**。
4. **`/audit/next-path`**：与真结局一同款的渐变文字演绎（含与 **林澈学长** 的对话 + 「下一段路径，由你开始」）。
5. 滚到底 **Stroll** → **`/audit/stroll`** 尾飞 → 邮箱感谢信。

> 备用：也可在 **`/audit/recruit`** 手动输入 `PATH-2034-LC`，但正式流程已改为提交后自动进入内推页，**无需再点招募页**。

## 十一、给新玩家的三条建议

1. **一次只搜一个词**，不要输入整句或多余符号；搜不到有时也是线索。
2. **对照四条核验线**：终端四条都亮后再提交，漏线容易误判程野性质。
3. **真结局一与二的差别**只在是否向 Ani 拿到 **`PATH-2034-LC`** 并在反思中选择「认可」——调查部分几乎相同，多走一条学生系统对话即可解锁更完整的「延续者」路线。
